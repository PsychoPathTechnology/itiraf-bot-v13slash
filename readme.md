## Kurulum
1. Eğer `.env` dosyası gözükmüyorsa bir `.env` dosyası oluşturun.
2. İçerisine aşağıdaki değerleri ekleyin:
   - **Token**: Discord botunuzun token değerini ekleyin.
   - **MongoDB**: Veritabanı bağlantı URI'nizi ekleyin.
